# InstaBot
Ever wanted to find out who unfollowed you on Instagram? Now you can!

## Dependencies:
- sys
- chromedriver (added to PATH and same version as Chrome)
- selenium
- python-dotenv
To install these dependencies, do:\
```pip3 install -r requirements.txt```

## To use it for yourself:
1. fork this repository
2. clone your repository using git\
```git clone https://github.com/<github-username>/InstaBot.git```
3. create a .env file with your necessary details (refer to .env.example)\
   -> IGN=<your_username>\
   -> PASS=<your_password>\
   -> USR=<account_name> (this can be the same as your username)
